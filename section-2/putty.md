### PuTTY

**PuTTY** is a free and open-source **terminal emulator, serial console, and network file transfer application**.  

- Commonly used to **connect to remote servers** using **SSH, Telnet, or serial connections**.  
- Allows users to **remotely manage servers** securely.  
- Lightweight and widely used on **Windows**, Linux, and macOS.  

**Example Uses:**  
- SSH into a Linux server from Windows  
- Configure network devices via Telnet/Serial  
- Transfer files using SCP (with PuTTY tools like PSCP)
